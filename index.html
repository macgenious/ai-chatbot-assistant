How this was made

Client-Side Security

This application follows a "Bring Your Own Key" architecture. Your API key is stored strictly in sessionStorage, ensuring it disappears when you close the tab.

Real-Time Streaming

Utilizing the Gemini API's streamGenerateContent endpoint, responses are decoded and rendered chunk-by-chunk for an immediate, conversational feel.

Modern UI/UX

Built with vanilla CSS using Glassmorphism principles, responsive flexbox layouts, and hardware-accelerated animations for a premium experience.
